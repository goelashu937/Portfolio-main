# Portfolio Project
This is a responsive portfolio site created with ReactJS and plain CSS that tells about me and implements Swiper for sliding components and EmailJS for contact services.

# Components
- About
- Skills
- Qualification
- Projects
- Position of Responsibility
- Certifications
- Contact me 

### Project Features:
- Responsive Layout
- Fixed Navigation
- Pop-up Modals
- Tab Panels
- Slider + Pagination
- Contact Form
- Scroll to Top
